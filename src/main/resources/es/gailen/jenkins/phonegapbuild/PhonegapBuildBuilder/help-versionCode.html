MUST BE A NUMBER. Version used for internal build tagging in both Apple and Google stores. Maps to 'versionCode' and 'CFBundleVersion' in config.xml. If not setted, the information from config.xml will be used. If setted, the config.xml will be modified in Jenkins' working directory. Beware: If you ask Phonegap Build plugin to override this field in www/config.xml, you better make sure your git or svn working copy in Jenkins gets resetted with every build.
